# Espresso Yourself POS System ☕🌿

A nature-inspired Point of Sale (POS) system designed for a mindful café experience. This system allows users to browse a coffee and food menu, search for specific items, manage an order summary, and generate printable receipts.

## ✨ Features
- **Dynamic Menu:** Automatically renders items from a data array.
- **Live Search:** Filter menu items by name or category in real-time.
- **Responsive Grid:** A clean, flexible layout that adapts to different screen sizes.
- **Cart Management:** Add items, adjust quantities, and remove items with live tax and total calculations.
- **Printable Receipts:** Generates a clean, professional receipt layout for printing.

## 🛠️ Technologies Used
- **HTML5:** Semantic structure.
- **CSS3:** Custom properties (CSS variables), Flexbox, and CSS Grid for the layout.
- **JavaScript (Vanilla):** Dynamic DOM manipulation, array filtering, and event handling.
- **Google Fonts:** 'Playfair Display' and 'Poppins' for a premium aesthetic.

## 🚀 How to Use
1. Clone the repository or download the `index.html` file.
2. Open `index.html` in any modern web browser.
3. Use the search bar to find items like "Forest Mocha".
4. Click "Add to Order" to populate the Order Summary.
5. Click "Print Receipt" to view the formal order breakdown.
